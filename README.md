# 🏨 Hospitality Analysis: Hotel Chain Business [Power BI | Excel]

![Project Thumbnail](hospitality_thumbnail.png)

## 📌 Project Overview

The **Atliq Grands** hotel chain faced declining revenue and weakening market presence. This project delivers a comprehensive **Power BI dashboard** and Excel-based analysis aimed at uncovering insights to drive a **20% increase** in both revenue and brand impact.

## 📊 Tools & Skills Used

- **Power BI**
- **Microsoft Excel**
- Data Cleaning
- KPI Analysis
- Dashboard Design
- Business Intelligence

## 🎯 Objectives

- Identify key performance metrics affecting revenue
- Benchmark performance across hotels and locations
- Provide actionable insights to decision-makers
- Streamline communication through visuals

## 📈 Key Insights Delivered

- Comparison of revenue trends across different hotel properties
- Occupancy rate breakdown by city and month
- High-performing vs low-performing departments
- Data-driven suggestions for improving efficiency
